## 🚀 Command Reference

| Command | Usage | Description |
| :--- | :--- | :--- |
| **`init`** | `./primer init` | Initialize your secure vault and establish the master key. |
| **`add`** | `./primer add <service>` | Securely store credentials for a new service via an interactive prompt. |
| **`get`** | `./primer get <service>` | Retrieve credentials. Displays the ID and copies the PW to the clipboard. |
| **`list`** | `./primer list` | View all registered service names within the vault. |
| **`remove`** | `./primer remove <service>` | Permanently delete a specific entry from your vault. |
| **`help`** | `./primer help` | Show detailed usage information and available flags. |
